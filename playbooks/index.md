# Playbooks

Real-world workflows using OpenCode and oMO. Each playbook is a step-by-step recipe — follow along with your own project.

## Available Playbooks

- [Start a New Project](/playbooks/new-project) — From idea to running code with /start-work
- [Add a Feature](/playbooks/add-feature) — Extend an existing project with Sisyphus
- [Debug a Bug](/playbooks/debug) — Reproduce, diagnose, fix with the pipeline
- [Refactor with Grill](/playbooks/refactor-grill) — Use grill-me to clarify intent before refactoring
