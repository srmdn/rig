## Share a Custom Skill

Created a useful skill with `/write-a-skill`? Share it:

1. Open a PR adding your skill to a `community/skills/` directory
2. Include a README explaining what it does and when to use it
3. Link it in the community config page

## Submission Channels

- **GitHub PR**  --  Add to `reference/examples/` or `community/skills/`
- **GitHub Discussion**  --  Start a discussion in the repo
- **Issue**  --  Open an issue with `[config-share]` or `[skill-share]` prefix
