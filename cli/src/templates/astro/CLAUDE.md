# CLAUDE.md — Astro Project

## Identity
- Astro framework, SSR mode
- Use `git config user.name "srmdn" && git config user.email "mail@saidwp.com"` before first commit

## Stack
- Framework: Astro
- Runtime: Bun (or Node)
- Content: Markdown / MDX
- Styling: CSS Modules or Tailwind
- Deployment: static or SSR (Node adapter)

## Conventions
- Pages in `src/pages/`
- Components in `src/components/`
- Layouts in `src/layouts/`
- Content collections in `src/content/`
- Static assets in `public/`

## Commands
- `bun run dev` — start dev server
- `bun run build` — production build
- `bun run preview` — preview production build
- `bunx astro check` — type checking

## Commit Style
- Small, atomic commits
- Conventional commit prefixes: feat, fix, refactor, docs, chore
